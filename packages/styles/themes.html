CSS code for custom theme

Default theme! No additional/custom styles.

For creating a new theme/updating theming features

Simply duplicate assets/css/blue-theme.scss and customize. It will generate the static CSS file with new values for custom properties.
For creating a previsualisation in the Design System, duplicate the button in this page and update values, and update hashes for CSP style-src.

Before committing new themes, do not forget to launch copy-theme-flat-css-files.sh, in order to have flat CSS files in the repo.

For a reference of all CSS custom properties used, have a look at Sass/CSS variables. If an element needs to use a custom property in order to be “themable”, please do update design-system-website/_themes.scss for reference.
