The application needs a .env file with the following environment variables:

- LIVEKIT_URL
- LIVEKIT_API_KEY
- LIVEKIT_API_SECRET
- LIVEKIT_ROOM_KEY

Currently having the following flags for experimental/prototype features in development:

- EXPERIMENTAL_FACE_CROP: automatic zoom and follow on the participant's face
- EXPERIMENTAL_ACTIVE_SPEAKER_ORDERING: bringing the active speakers forward, this is going to be revised
