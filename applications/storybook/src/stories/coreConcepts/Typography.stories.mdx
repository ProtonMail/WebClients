# Typography

## Normal text

Proton apps suite uses a system font stack. System fonts are the default fonts used by any given operating system, helping performance by eliminating the need to download font files.

```css
body {
    font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen-Sans, Ubuntu, Cantarell, 'Helvetica Neue', sans-serif;
}
```

## Source code

For any source code, Proton apps suite uses a monospaced system font stack. Why double `monospace`? [Here is the explanation](https://github.com/necolas/normalize.css/blob/master/README.md#pre-code-kbd-samp).

```css
pre, code, kbd, samp {
    font-family: SFMono-Regular, Consolas, 'Liberation Mono', 'Menlo', monospace, monospace;
}
```
