## Source code

For any source code, Proton apps suite uses a monospaced system font stack.

```css
pre, code, kbd, samp {
    font-family: ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, 'Liberation Mono', 'Courier New', monospace;
}
```
