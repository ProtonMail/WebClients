## Items size and behaviour


<div class="flex flex-justify-space-between mb2 on-mobile-flex-column">

    <div class="flex border color-global-light">
      <div class="bg-primary p1 w150p">.w150p</div>
      <span class="bg-primary p1 w25">.w25</span>
      <em class="bg-primary p1 flex-item-fluid">.flex-item-fluid</em>
    </div>
  </div>
  <div class="w49 flex-align-self-center">
<Source
  language='html'
  light
  format
  code={`
<div className="flex ">
  <div className="w150p">.w150p</div>
  <span className="w25">.w25</span>
  <em className="flex-item-fluid">.flex-item-fluid</em>
</div>
`}
/>
  </div>
</div>

<p><code>flex-item-fluid</code> applies <code>flex: 1 1 0px</code> to the element.</p>
<p><code>flex-item-fluid-auto</code> applies <code>flex: 1 1 auto</code> to the element.</p>

<p>If you need an element that <strong>does not shrink</strong>, you may use <code>flex-item-noshrink</code> (applies <code>flex-shrink: 0</code>).<br />
If you need an element that <strong>does not grow</strong>, you may use <code>flex-item-nogrow</code> (applies <code>flex-grow: 0</code>).<br />
If you need an element that <strong>does not flex at all</strong>, you may use <code>flex-item-noflex</code> (applies <code>flex: 0</code>).<br />
If you need an element that <strong>centers itself</strong>, you may use <code>flex-align-self-center</code> (applies <code>align-self: center</code>). In some other cases, you may use <code>mauto</code> class (<code>margin: auto</code>).
<br />
If you need an element that <strong>aligns itself on the top/beginning</strong>, you may use <code>flex-align-self-start</code> (applies <code>align-self: flex-start</code>), same goes with <code>flex-align-self-end</code>.
</p>
<p><code>flex-item-grow-2</code> allows the fluid item to grow per 2 factor.</p>
<p><code>flex-flex-children</code> applies <code>display: flex</code> to all childrens.</p>

## Responsive

      <em class="bg-primary p1 w30">.w30</em>
