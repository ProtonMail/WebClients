## Margin auto

<ul>
    <li>
        <code>.mtauto</code> = <code>margin-top: auto</code>.
    </li>
    <li>
        <code>.mrauto</code> = <code>margin-right: auto</code>.
    </li>
    <li>
        <code>.mbauto</code> = <code>margin-bottom: auto</code>.
    </li>
    <li>
        <code>.mlauto</code> = <code>margin-left: auto</code>.
    </li>
    <li>
        <code>.mauto</code> = <code>margin: auto</code>.
    </li>
    <li>
        <code>.mxauto</code> = <code>.mlauto</code> + <code>.mrauto</code>
    </li>
    <li>
        <code>.myauto</code> = <code>.mtauto</code> + <code>.mbauto</code>
    </li>
</ul>

Use these classes to position elements [with flexbox](?path=/docs/css-utilities-legacy-flexbox--page).
