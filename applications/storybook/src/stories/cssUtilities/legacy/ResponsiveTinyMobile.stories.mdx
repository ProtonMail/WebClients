Other flexbox helpers for tinymobile:

-   <code>on-tiny-mobile-flex-align-self-start</code> applies <code>align-self: start</code> to the element for mobile and
    smaller breakpoints.

## Margin/padding helpers

Please check before how <a href="?path=/docs/front-end-helpers-margin-padding--page">margin/padding helpers</a> work.

-   <code>.on-tiny-mobile-m</code> = <code>margin</code>
-   <code>t/r/b/l</code> = <code>top/right/bottom/left</code>
-   <code>0/1</code> = 0/1em

Examples:

<div class="flex flex-justify-space-between mb-8 on-mobile-flex-column">
    <div class="w49 mb-4 md:mb-0 border">
        <div class="on-tiny-mobile-pl1 on-tiny-mobile-pr1 m-4 sm:m-0 bg-primary">
            <code>.on-tiny-mobile-m1</code> = <code>margin: 1em</code> <span class="color-white">on tinymobile</span>.
        </div>
    </div>
</div>
