## Margin auto

<ul>
    <li>
        <code>.mt-auto</code> = <code>margin-top: auto</code>.
    </li>
    <li>
        <code>.mr-auto</code> = <code>margin-right: auto</code>.
    </li>
    <li>
        <code>.mb-auto</code> = <code>margin-bottom: auto</code>.
    </li>
    <li>
        <code>.ml-auto</code> = <code>margin-left: auto</code>.
    </li>
    <li>
        <code>.m-auto</code> = <code>margin: auto</code>.
    </li>
    <li>
        <code>.mx-auto</code> = <code>.ml-auto</code> + <code>.mr-auto</code>
    </li>
    <li>
        <code>.my-auto</code> = <code>.mt-auto</code> + <code>.mb-auto</code>
    </li>
</ul>

Use these classes to position elements [with flexbox](?path=/docs/css-utilities-flexbox--page).

## Responsive

The new helper classes are **all** available to use without media query, as well as `sm`, `md`, `lg` and `xl` variants. This allows to tweak spacings precicely for every breakpoint if needed. The blue square in the demo area underneath has the classes `ml-4 sm:ml-8 md:ml-10 lg:ml-12 xl:ml-14` which translates to margin-left 4 (16px) on `tinymobile`, 8 (32px) on `mobile`, 10 (40px) on `tablet`, 12 (48px) on `desktop` and 14 (56px) on `extra large`. The new system works **mobile first**, that means the base `ml-4` will be overridden by `lg:ml-12` on larger screens.
