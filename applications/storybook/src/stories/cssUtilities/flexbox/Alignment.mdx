# Alignment

## Justify start

Flex items are aligned at the start of the flex container, depending on the writing-mode.

`className='flex justify-start'`

    <Story id="css-utilities-flexbox-alignment--justify-start" />

## Justify center

## Justify end

Flex items are aligned towards the end of the flex container, depending on the writing-mode.

## Align items at the start in a column

When using `.flex-column`, use `.items-start` to align the items at the start of the flex container, depending on the writing-mode.

`className='flex flex-column items-start'`

<Canvas>
    <Story id="css-utilities-flexbox-alignment--items-start-column" />
</Canvas>

## Center items in a column

When using `.flex-column`, use `.items-center` to center the items in the flex container.

`className='flex flex-column items-center'`

    <Story id="css-utilities-flexbox-alignment--items-center-column" />
