# Responsive

Many (but not all) flex helper classes can be prefixed with `on-mobile` or `on-tablet`

## Column on mobile

This will display items in a column on mobile, but not on tablet or desktop (resize preview to see effect)

`className='flex on-mobile-flex-column'`

<div className="border p-2 rounded bg-weak" style={{ width: '320px', resize: 'both', overflow: 'auto' }}>
    <Story id="css-utilities-flexbox-responsive--column-mobile" />
</div>

## Column on tablet

This will display items in a column on mobile and tablet, but not on desktop (resize preview to see effect) `className='flex on-tablet-flex-column'`

<div className="border p-2 rounded bg-weak" style={{ width: '900px', resize: 'both', overflow: 'auto' }}>
    <Story id="css-utilities-flexbox-responsive--column-tablet" />
</div>

## Remove gap on mobile

This will remove the gap on mobile, but not tablet or desktop `className='flex gap-0 md:gap-4'`

<div className="border p-2 rounded bg-weak mb-4" style={{ width: '320px', resize: 'both', overflow: 'auto' }}>
    <Story id="css-utilities-flexbox-responsive--flex-gap-0" />
</div>

<div className="border p-2 rounded bg-weak" style={{ width: '750px', resize: 'both', overflow: 'auto' }}>
    <Story id="css-utilities-flexbox-responsive--flex-gap-0" />
</div>
