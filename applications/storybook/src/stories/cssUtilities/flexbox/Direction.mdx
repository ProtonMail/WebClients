# Flexbox

This defines a flex container and enables a flex context for all its direct children. This class is mandatory when using any of the flex helper classes.

`className='flex'`

<Canvas withSource="true">
    <Story id="css-utilities-flexbox-direction--flex" />
</Canvas>

Note: flex class applies `min-width: 0` and `min-height: 0` to the direct children. In some very super-rare cases (selects, etc.), this can create bugs. In these cases, you may use `flex-no-min-children`.

## Flex direction

This defines the directions that the flex items are placed

-   `row` (default): left to right in ltr; right to left in rtl
-   `row-reverse`: right to left in ltr; left to right in rtl
-   `column`: same as row but top to bottom
-   `column-reverse`: same as row-reverse but bottom to top

## Flex Wrap

When using the `.flex` helper class, items will automatically flow into the next line. (due to `flex-wrap: wrap`) To always keep items on one line, use `flex-nowrap`.

`className='flex flex-nowrap'`
