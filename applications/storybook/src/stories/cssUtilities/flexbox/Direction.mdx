# Flex direction

This defines the directions that the flex items are placed

- `row` (default): left to right in ltr; right to left in rtl
- `row-reverse`: right to left in ltr; left to right in rtl
- `column`: same as row but top to bottom
- `column-reverse`: same as row-reverse but bottom to top

## Flex Wrap

When using the `.flex` helper class, items will automatically flow into the next line. (due to `flex-wrap: wrap`) To always keep items on one line, use `flex-nowrap`.

`className='flex flex-nowrap'`
