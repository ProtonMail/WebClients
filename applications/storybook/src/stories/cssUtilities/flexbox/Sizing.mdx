# Item Sizing

Use `.flex-item-fluid` or `.flex-item-fluid-auto` on flex items to set the sizing behaviour.

**Resize containers to see the effect.**

`.flex-item-fluid` will take as little space as it's content allows (will wrap text).

<div className="border p-2 rounded bg-weak" style={{ width: '320px', resize: 'both', overflow: 'auto' }}>
    <Story id="css-utilities-flexbox-sizing--flex-item-fluid" />
</div>

`.flex-item-fluid-auto` will take as little space as it's content allows (will not wrap text).

<div className="border p-2 rounded bg-weak" style={{ width: '320px', resize: 'both', overflow: 'auto' }}>
    <Story id="css-utilities-flexbox-sizing--flex-item-fluid-auto" />
</div>

`.flex-item-grow-2` will grow by factor 2.

<div className="border p-2 rounded bg-weak" style={{ width: '320px', resize: 'both', overflow: 'auto' }}>
    <Story id="css-utilities-flexbox-sizing--flex-item-grow" />
</div>

`.flex-item-noshrink` This item will not shrink and not wrap text.

<div className="border p-2 rounded bg-weak" style={{ width: '320px', resize: 'both', overflow: 'auto' }}>

`.flex-item-nogrow` This item will shrink but it will not grow.
