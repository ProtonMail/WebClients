# Flexbox

This defines a flex container and enables a flex context for all its direct children. This class is mandatory when using any of the flex helper classes.

`className='flex'`

<Canvas withSource="true">
    <Story id="css-utilities-flexbox-flex--flex" />
</Canvas>

Note: flex class applies `min-width: 0` and `min-height: 0` to the direct children. In some very super-rare cases (selects, etc.), this can create bugs. In these cases, you may use `flex *:min-size-auto`.
