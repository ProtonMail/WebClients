### Responsive

Every custom sizing utilities exist with responsive prefixes, as well as their dedicated variable. This way, you can use several custom sizing responsive utilities together.

Where responsive prefixes are separated by colons (:) in a class name, they are separated by a dash (-) in the name of a dedicated variable.

<!--- prettier-ignore-start --->
```html
<div
    className="
        min-w-custom
        max-w-custom
        md:min-w-custom
        md:max-w-custom
        w-custom
        sm:w-custom
        md:w-custom
        lg:w-custom
        h-custom
        lg:h-custom
    "
    style={{
        '--min-w-custom: '2rem',
        '--max-w-custom: '40rem',
        '--md-min-w-custom: '5rem',
        '--md-max-w-custom: '60rem',
        '--w-custom': '22%',
        '--sm-w-custom': '37%',
        '--md-w-custom': '53%',
        '--lg-w-custom': '82%',
        '--h-custom': '5rem',
        '--lg-h-custom': '15rem',
    }}
/>
```
<!--- prettier-ignore-end --->
