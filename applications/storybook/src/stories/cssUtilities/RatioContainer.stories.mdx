# Square and ratio helpers

For some components/special cases (responsive <code>iframe</code>, making a responsive square shape, responsive typography with SVG, etc.), we have to use the “padding-hack” technique, which is basically:

    <div class="w-1/2 self-center">
        <ul>
            <li>A container uses padding-hack to get the desired ratio</li>
            <li>And content inside is positioned in absolute and takes all available space.</li>
        </ul>
    </div>
    <div class="w-1/2">
        <Source
            language="html"
            light
            format
            code={`
<div class="ratio-square p-4">
    Some content in a square box
    with ratio-square class.
</div>
`}
        />
    </div>
</div>
