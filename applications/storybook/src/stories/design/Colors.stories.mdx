Class aliases exist also for <code>$primary-color</code>: <code>fill-primary</code>/<code>bg-primary</code>/<code>color-primary</code>.

For highlighting using <code>bg-weak</code>, you may use the alias <code>bg-global-highlight</code>, same for <code>bg-white/bg-white-dm</code>, <code>bg-global-muted/bg-global-muted-dm</code>, <code>color-global-grey/color-global-grey-dm</code> (dark mode friendly aliases). <code>bg-global-border</code> is already dark-mode compatible.
