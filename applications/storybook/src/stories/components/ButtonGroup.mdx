# ButtonGroup

The button group is simply a group of [Button](?path=/docs/components-button--basic) components.

## How to use

```html
<ButtonGroup>
    <Button>regular item</Button>
    <Button>regular item</Button>
    <Button className="is-selected">selected item</Button>
</ButtonGroup>
```

## Variants

The ButtonGroup component demands its own `shape`, `color` and `size` parameters to adapt its embeded buttons automatically.
