# Icon

Icon component can have different sizes, configurable via the `size` prop, 16 pixels like by default. They are colored via text color helpers and inherit the parent container's text color by default.

All the available icons in our icon set are reachable in the [icon dedicated documentation page](/docs/core-concepts-icons--icons).

## Sizing

There's a sizing taxonomy to be used in the `size` prop. This taxonomy is used in the [regular sizing system as well](/docs/css-utilities-sizing--fractions#framework), but with less values for now.

Basically, the base unit is `4`, so if you want an icon to be displayed at a size of a 20 pixels square, divide this value by the base unit, and you have your `size` value, so here `5`.
