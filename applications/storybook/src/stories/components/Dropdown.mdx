# Dropdown

`Dropdown` is a component that can be used to display content on top of other content. It can be used together with:

- `DropdownButton` (button with a caret)
- `DropdownMenu`
- `DropdownMenuButton`
- `SimpleDropdown` (wraps Dropdown and DropdownButton)
