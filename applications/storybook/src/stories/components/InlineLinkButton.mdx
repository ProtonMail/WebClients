# InlineLinkButton

The InlineLinkButton component is a button with the exact look and behaviour of a link.

<Alert type="info">
    <b>The InlineLinkButton should only be used inside a text content.</b> If you need a standalone underline button, or placed next to an input field or another button, please refer to the <a href="?path=/docs/components-button--basic">button with underline shape</a>.
</Alert>
