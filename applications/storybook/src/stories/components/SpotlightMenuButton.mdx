# SpotlightMenuButton

Just like Spotlight, it allows to welcome the user with a dedicated Dropdown containing a clickable menu like a guide, connected to a functionality. Difference with Spotlight is that the user can open and close the menu via a button at will.

It also comes with a pre-built components of menu items.
