# Sizing helpers

Here is the list of available sizing helpers:

<ul class="list-2columns on-tablet-list-1column">
  <li><code>.w0 </code> = <code> width: 0;</code></li>
  <li><code>.w1 </code> = <code> width: 1%;</code></li>
  <li><code>.w2 </code> = <code> width: 2%;</code></li>
  <li><code>.w3 </code> = <code> width: 3%;</code></li>
  <li><code>.w5 </code> = <code> width: 5%;</code></li>
  <li><code>.w10 </code> = <code> width: 10%;</code></li>
  <li><code>.w15 </code> = <code> width: 15%;</code></li>
  <li><code>.w20 </code> = <code> width: 20%;</code></li>

  <li><code>.w33 </code> = <code> width: 33.333%;</code></li>
  <li><code>.w40 </code> = <code> width: 40%;</code></li>
  <li><code>.w45 </code> = <code> width: 45%;</code></li>
  <li><code>.w49 </code> = <code> width: 49%;</code></li>
  <li><code>.w50 </code> = <code> width: 50%;</code></li>

  <li><code>.w66 </code> = <code> width: 66.666%;</code></li>

  <li><code>.w100 </code> = <code> width: 100%;</code></li>
  <li><code>.max-w50 </code> = <code> max-width: 50%;</code></li>
  <li><code>.max-w60 </code> = <code> max-width: 60%;</code></li>
  <li><code>.max-w80 </code> = <code> max-width: 80%;</code></li>
  <li><code>.max-w100 </code> = <code> max-width: 100%;</code></li>
  <li><code>.h100 </code> = <code> height: 100%;</code></li>
  <li><code>.h100v </code> = <code> height: 100vh;</code></li>
</ul>

Some other are available, please check in your Sass files the ones used, you can define Sass variables to generate the ones your need:

<ul class="list-2columns on-tablet-list-1column">
  <li><code>$list-width-px</code></li>
  <li><code>$list-width-rem</code></li>
  <li><code>$list-width-em</code></li>
  <li><code>$list-max-widths</code><br /> (ex.: <code>.max-w100 </code> = <code>max-width: 100%</code>)</li>
  <li><code>$list-max-width-px</code></li>
  <li><code>$list-max-width-em</code><br />  (ex.: <code>.max-w15e </code> = <code>max-width: 15em;</code>)</li>
  <li><code>$list-min-width-em</code></li>
</ul>
