## Dark Mode

For React applications, a <code>isDarkMode</code> (<code>appConfigBody-darkmode</code> for Angular) class is applied to the <code>body</code> when dark mode is triggered.

Then you have two helpers: <code>hide-on-darkmode</code> will hide something when dark mode is triggered, and <code>display-on-darkmode</code> (which is hidden by default) will be displayed when dark mode is triggered.
