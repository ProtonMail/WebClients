If you want to apply <code>text-ellipsis</code> and <code>pre</code> classes, you need to wrap your content like this:

<div class="flex flex-justify-space-between mb2 on-mobile-flex-column">
  <div class="w49 flex-align-self-center">
    <span class="text-ellipsis inline-block max-w100p"><span class="text-pre">Lorem    ipsum dolor sit amet</span></span><br />
    To be used with <code>max-w100</code> and <code>inline-block</code> (or with flexbox helpers).
  </div>
  <div class="w49">
<Source
  language='html'
  dark
  format
  code={`
<span className="text-ellipsis">
  <span className="text-pre">
  Lorem    ipsum dolor sit amet
  </span>
  </span>
`}
/>

  </div>
</div>

These helpers may be prefixed by <code>on-tablet-/on-mobile-</code> to apply them only for tablets/mobiles.
