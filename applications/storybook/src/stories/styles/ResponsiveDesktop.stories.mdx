# Responsive - Desktop

## Breakpoints

Values are given in “pixels” (in standard cases: with base font size to <code>16px</code>), but breakpoints are computed into <code>em</code>, which helps to better manage zoom. (for example, “mobile” breakpoint can be triggered on a large screen if you zoom a lot)

The current active breakpoint value is available in <code>body::before content</code> value (for major breakpoints), if you need to know (from JavaScript pov) which major breakpoint is active, use the <code>useActiveBreakpoint.ts</code> helper in <code>react-components</code>.
