# Margin/padding helpers

Here is a set of margin and padding helpers - check in Design system fragments the values you have.

- <code>.m</code> = <code>margin</code>
- <code>.p</code> = <code>padding</code>
- <code>t/r/b/l</code> = <code>top/right/bottom/left</code>
- <code>0/0-5/1/2</code> = 0/0.5em/1em/2em

Examples:

Also available:

<ul>
  <li><code>.mtauto</code> = <code>margin-top: auto</code>.</li>
  <li><code>.mrauto</code> = <code>margin-right: auto</code>.</li>
  <li><code>.mbauto</code> = <code>margin-bottom: auto</code>.</li>
  <li><code>.mlauto</code> = <code>margin-left: auto</code>.</li>
  <li><code>.mauto</code> = <code>margin: auto</code>.</li>
  <li><code>.center</code> = <code>.mlauto</code> + <code>.mrauto</code></li>
</ul>

This helps for positionning blocks or [with flexbox](?path=/docs/front-end-helpers-flexbox--page).

For some very rare cases, we also have helpers in <code>rem</code>: <code>.mt1r</code>, <code>mt0-5r</code>, <code>mr0-5r</code>.
