# MISC: CSS table/Ratio container

## Table-design using CSS


- <code>.row</code> = <code>display: table; table-layout: fixed </code>.
- <code>.inline-row</code> = <code>display: inline-table; table-layout: fixed </code>.
- <code>.line</code> = <code>display: table-row</code>.
- <code>.col</code> = <code>display: table-cell; vertical-align: top</code>.
- <code>.col-no-align</code> = <code>display: table-cell</code>.

- <code>.align-top</code> = <code>vertical-align: top</code>.
- <code>.align-bottom</code> = <code>vertical-align: bottom</code>.
- <code>.align-middle</code> = <code>vertical-align: middle</code>.
- <code>.align-sub</code> = <code>vertical-align: sub</code>.
- <code>.align-baseline</code> = <code>vertical-align: baseline</code>.

These vertical alignments also work for simple texts or inline-positionned stuff.
