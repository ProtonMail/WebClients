## Linearizing content on tinymobile

<div class="flex flex-justify-space-between mb2 on-mobile-flex-column">
  <div class="w49 flex-align-self-center">
	Applies <code>float: none; display: block; width: auto;</code> to the element for tinymobile breakpoint.
  </div>
  <div class="w49">
<Source
  language='html'
  dark
  format
  code={`
<span className="auto-tiny-mobile">…</span>
`}
/>
  </div>
</div>

<br />

<strong>Flexbox version:</strong> resize the window, by default, elements take all the width on mobile thanks to <code>on-tiny-mobile-flex-column</code>.

<div class="flex flex-justify-space-between mb2 on-tiny-mobile-flex-column">
  <div class="w49 flex-align-self-center">
    <div class="flex flex-justify-space-between on-tiny-mobile-flex-column bordered color-global-light">
      <div class="bg-primary p1 w30">.w30</div>
      <span class="bg-primary p1 w30">.w30</span>
      <em class="bg-primary p1 w30">.w30</em>
    </div>
  </div>
  <div class="w49 flex-align-self-center">
<Source
  language='html'
  dark
  format
  code={`
<div className="flex flex-justify-space-between on-tiny-mobile-flex-column">
  <div className="w30">div</div>
  <span className="w30">span</span>
  <em className="w30">em</em>
</div>
`}
/>

  </div>
</div>

Other flexbox helpers for tinymobile:

- <code>on-tiny-mobile-flex-align-self-start</code> applies <code>align-self: start</code> to the element for mobile and smaller breakpoints.



## Margin/padding helpers

Please check before how <a href="?path=/docs/front-end-helpers-margin-padding--page">margin/padding helpers</a> work.

- <code>.on-tiny-mobile-m</code> = <code>margin</code>
- <code>t/r/b/l</code> = <code>top/right/bottom/left</code>
- <code>0/1</code> = 0/1em

Examples:

<div class="flex flex-justify-space-between mb2 on-mobile-flex-column">
  <div class="w49 on-mobile-mb1 bordered">
    <div class="on-tiny-mobile-pl1 on-tiny-mobile-pr1 on-tiny-mobile-m1 bg-primary">
      <code>.on-tiny-mobile-m1</code>  = <code>margin: 1em</code> <span class="color-white">on tinymobile</span>.
    </div>
  </div>
</div>
