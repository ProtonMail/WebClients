5. SPF Recommended

SPF is used to specify who is allowed to send email for the domain so we strongly recommend including ProtonMail in your SPF record. Please add the following TXT record into your DNS. This can typically be done in the control panel of your domain name registrar.

Please add the following TXT record. Note, DNS records can take several hours to update.

Learn more

Type Host name Value / Data / Points to
TXT @ v=spf1 include:_spf.protonmail.ch mx ~all
